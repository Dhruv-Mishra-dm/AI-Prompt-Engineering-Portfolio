# AI Content Generation Suite

This directory contains advanced prompt engineering systems designed to automate, optimize, and standardize text generation. The core objective of these architectures is to move beyond basic conversational AI and build reliable engines for marketing, technical documentation, and cross-platform content strategy.

## Core Competencies Demonstrated
* **Zero-Shot Prompting:** Designing robust instructions that generate highly specific formats without requiring prior examples.
* **Few-Shot Prompting (Voice Cloning):** Reverse-engineering style, tone, and syntax from sample data to perfectly replicate specific personas.
* **Systematic Extraction:** Utilizing task decomposition to filter unstructured "noise" and format raw data into structured, professional documentation.

---

## 📂 Project Index

### 1. [Multi-Platform Content Repurposer](./Multi-Platform-Content-Repurposer.md)
A zero-shot engine designed to transform technical video transcripts or raw notes into platform-optimized content for LinkedIn, Twitter, and SEO blogs. It incorporates strict guardrails to prevent hallucination while adapting the format and tone to fit the specific constraints of each social platform.

### 2. [Brand Voice Cloner](./Brand-Voice-Cloner.md)
A few-shot prompting system that extracts the stylistic "DNA" from provided writing samples. It allows the AI to dynamically pivot and generate new content in highly specific, un-robotic tones (e.g., Aggressive Tech Visionary, Gen-Z Marketing Guru).

### 3. [Technical Changelog Normalizer](./Technical-Changelog-Generator.md)
An extraction engine that uses role assignment and multi-step logic to filter unstructured, messy developer notes. It strips out chatter and debug logs, classifying the remaining data into a clean, machine-readable Markdown table of New Features, Bug Fixes, and Deprecations.
