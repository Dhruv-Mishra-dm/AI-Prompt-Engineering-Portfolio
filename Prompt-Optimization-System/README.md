# Prompt Optimization & Guardrail Systems

This directory focuses on the technical control of LLMs, specifically addressing hallucination prevention, structured output enforcement, and objective reasoning.

### Projects in this folder:
1. **The Hallucination Guardrail**: A fact-checking engine designed to prevent AI fabrications.
2. **Strict JSON Enforcer**: A system to force LLMs into machine-readable formats for software integration.
3. **The Unbiased Summarizer**: A prompt architecture designed to strip emotional bias from controversial texts.
