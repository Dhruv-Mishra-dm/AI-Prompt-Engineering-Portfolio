## 🛠️ Methodologies & Prompting Techniques
Across this repository, I utilize industry-standard prompt engineering techniques:
* **Constraint-Based Prompting:** Using strict negative constraints to prevent out-of-bounds knowledge retrieval and formatting errors.
* **Structured Output Enforcement:** Bounding LLM responses into strict Markdown tables or valid JSON objects.
* **Task Decomposition:** Breaking down complex extraction or analytical tasks into multi-step, logic-gated pipelines.
* **Socratic & Role Prompting:** Narrowing the probability space by forcing the AI into highly specific, professional personas (e.g., C Memory Debugger, Objective Analysis Engine).
